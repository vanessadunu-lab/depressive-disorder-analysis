# depressive-disorder-analysis
This project explores mental illness from 1990-2019 using country-level mental health data. This analysis serves to compare mental health data, specifically depressive disorder prevalence, across countries and determine changes or lack of change displayed by the data. 

## Objective
Analyze depressive disorder prevalence across countries from 1990-2019.

## Tools
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Questions Explored
- Which countries had the highest prevalence?
- Which countries had the lowest prevalence?
- How did prevalence change over time?

## Key Findings
- Prevalence varied across countries.
- Uganda showed higher prevalence than the selected comparison countries.
- Trends differed over time across countries.
